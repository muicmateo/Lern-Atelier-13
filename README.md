# Lern-Atelier-13

## LP13 – Phaser.js

### Entscheidung
Ich habe mich für **Phaser.js** entschieden, da es ideal für 2D-Web-Spiele ist und mächtige Tools für Physik und Tilemaps bietet.

### Projekt
Ich werde probieren, einen **Dungeon Crawler** zu erstellen.

### Arbeitspakete für 16.01
1. - [ ] Laden einer Dungeon-Karte (JSON), um die Spielwelt visuell darzustellen.
2. - [ ] Implementierung der Steuerung, um den Charakter flüssig durch den Dungeon zu bewegen.
3. - [ ] Wände unpassierbar machen, um die grundlegende Spielmechanik zu festigen.
4. - [ ] Kamera-Follow einrichten, damit die Sicht immer auf dem Spieler bleibt.
